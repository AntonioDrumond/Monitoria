# Estruturas dinâmicas: Pilha e Fila

### Estrutura básica: Célula
+ A célula é uma classe/objeto que contem dados e um ou mais ponteiros para outras células.
+ Estruturas dinâmicas são maneiras diferentes de organizar essas células e ponteiros

## Pilha
+ FILO -> First In Last Out -> Primeiro que entra, ultimo a sair
	+ Inserir e remover elementos do topo.
+ Contém um único ponteiro para o topo da pilha

## Fila
+ FIFO -> First In First Out -> Primeiro que entra, primeiro a sair
	+ Remove do final, insere no início
+ Contém um ponteiro para o início e um para o final
